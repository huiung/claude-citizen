# Kenney Sci-fi Sounds

Selected one-shot event sounds from Kenney's Sci-fi Sounds pack.

- Source: https://kenney.nl/assets/sci-fi-sounds
- Mirror: https://opengameart.org/content/sci-fi-sounds
- License: Creative Commons CC0
- Attribution: not required; crediting "Kenney.nl" is appreciated by the author.

This project uses only short event cues from the pack. Continuous engine, boost,
mining, and quantum-drive audio remain procedural WebAudio so they can react to
player input every frame.
